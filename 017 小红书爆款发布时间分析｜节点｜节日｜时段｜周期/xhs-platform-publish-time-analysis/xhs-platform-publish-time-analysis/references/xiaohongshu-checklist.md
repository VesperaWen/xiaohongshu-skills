# 小红书爆款发布时间分析｜节点｜节日｜时段｜周期 检查清单

## 任务定位

- 分类：01 爆款分析与对标研究
- 重点：找爆款、拆爆款、提炼模板、迁移选题、判断失败原因
- 适合场景：判断什么时候发、节日节点是否影响爆发、重发策略。
- 推荐输出：发布时间建议

## 输入字段

- 账号：昵称、定位、赛道、粉丝量、主页、置顶笔记、过往爆文。
- 用户：目标人群、痛点、消费能力、搜索习惯、评论高频问题。
- 内容：选题、标题、封面、正文、标签、视频脚本、直播脚本或达人brief。
- 数据：曝光、点击、点赞、收藏、评论、关注、私信、商品点击、成交。
- 参考：竞品账号、对标笔记、爆款截图、评论区、商品页或投放素材。

## 爆款拆解框架

1. 选题：是否踩中强需求、强情绪、强场景或强搜索。
2. 封面：缩略图是否一眼知道主题和好处。
3. 标题：是否包含关键词、痛点、数字、反差、身份或结果。
4. 正文：开头是否留人，中段是否有信息密度，结尾是否引导互动。
5. 评论：用户在问什么、反驳什么、想要什么、下一篇能写什么。
6. 转化：是否自然承接到关注、收藏、私信、商品、店铺或直播。

## 发布前检查

- 标题和封面是否同一个承诺。
- 正文是否兑现封面标题的承诺。
- 标签是否覆盖关键词、场景词、人群词和品类词。
- 是否有未经授权图片、字体、音乐、品牌Logo或达人素材。
- 是否有绝对化、功效化、虚假体验、诱导导流或平台敏感表达。

## 复盘模板

- 表现最好的一项：选题、封面、标题、正文、评论还是发布时间。
- 表现最弱的一项：点击差、互动差、收藏差、转粉差还是转化差。
- 可以保留的结构：标题公式、封面结构、正文模块、评论引导。
- 下一轮测试：只改一个主要变量，记录发布时间和数据变化。
